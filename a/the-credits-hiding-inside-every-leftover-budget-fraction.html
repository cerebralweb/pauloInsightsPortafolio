THE CREDITS HIDING INSIDE EVERY LEFTOVER BUDGET FRACTION

In this article ❧ Teams rarely report what they spent. They report what is left. A balance, a remaining quota, a fraction of a quota still open. To compare effort or cost across people, a manager has to flip each leftover into a used amount first. This problem shows how converting a remaining fraction into a consumed fraction, then summing across several accounts, turns scattered balances into one total a manager can plan against.

The Problem

Jorge, Sara, and Marta each manage a cloud project. Each project started the month with a 48 credit compute budget. At month end, Jorge's budget was 3/4 unused, Sara's was 2/3 unused, and Marta's was 1/6 unused.

How many credits did the three projects consume in total?

  • a) 88
  • b) 66
  • c) 15
  • d) 68
Click to Reveal the Answer

Correct Answer: d) 68 ✅

Math Solution

For each project, subtract the unused fraction from one whole to get the used fraction, then multiply by the 48 credit budget. Add the three results to get the total consumed.

The report gives what stayed, so the first move is always one minus the leftover.

Step-by-Step Breakdown

The following sequence shows how the final amount is derived:

  1. Jorge used: 1 minus 3/4 is 1/4, and 1/4 of 48 is 12
  2. Sara used: 1 minus 2/3 is 1/3, and 1/3 of 48 is 16
  3. Marta used: 1 minus 1/6 is 5/6, and 5/6 of 48 is 40
  4. Total: 12 plus 16 plus 40 is 68

Historical Background

Splitting a whole into parts is one of the oldest accounting tasks. Egyptian scribes worked almost entirely in unit fractions, parts with one on top, and used them to divide grain, land, and labor among many claimants. Merchants and tax officials across the ancient world reasoned the same way the manager does here, by tracking the part of an allotment still owed or still held and adding those parts to learn the whole. The arithmetic that converts a remaining share into a consumed share predates the symbols we now write it with.

Python Version

The script turns one leftover fraction into the credits a project used and reuses that rule for all three accounts, so it works for any budget and any fractions.

# What this does: works out the total credits three projects used in a month.
# Where it came from: three cloud projects each had the same credit budget and reported the part left over.
# The math idea: turn each leftover part into a used part, take that part of the budget, then add the three.
# You type in: the budget, and each project's leftover fraction as a top and bottom number.
# It prints: the total credits the three projects used.

# This function turns one leftover fraction into the credits that project used.
def credits_used(budget, top, bottom):
    # The leftover part of the budget, as a number between 0 and 1.
    leftover_part = top / bottom
    # The used part is everything that is not leftover.
    used_part = 1 - leftover_part
    # Take the used part of the budget.
    return budget * used_part

# Tell the reader what we will figure out.
print("Let's find the total credits three projects used.")

# Ask for the budget each project started with.
budget = float(input("Starting budget per project: "))

# Ask for Jorge's leftover fraction.
jorge_top = float(input("Jorge leftover top number: "))
jorge_bottom = float(input("Jorge leftover bottom number: "))

# Ask for Sara's leftover fraction.
sara_top = float(input("Sara leftover top number: "))
sara_bottom = float(input("Sara leftover bottom number: "))

# Ask for Marta's leftover fraction.
marta_top = float(input("Marta leftover top number: "))
marta_bottom = float(input("Marta leftover bottom number: "))

# Work out the credits each project used.
jorge_used = credits_used(budget, jorge_top, jorge_bottom)
sara_used = credits_used(budget, sara_top, sara_bottom)
marta_used = credits_used(budget, marta_top, marta_bottom)

# Add the three amounts.
total_used = jorge_used + sara_used + marta_used

# Show a whole number when the result has no fraction left.
total_used = int(total_used) if total_used.is_integer() else total_used

# Print the answer with words that fit the scenario.
print("Total credits the three projects used:", total_used)

Run it in a terminal, type 48 for the budget and each leftover as a top and bottom number, and it returns 68 for this case.

# Total Credits Used From Leftovers

This skill takes a shared starting budget and a leftover fraction for each of three projects, turns each leftover into the part that was used, applies that part to the budget, and adds the three results into one total. It takes the budget and three leftover fractions as input. It produces one number: the total credits the three projects used.

---

## Dependencies

No third party libraries are needed. For exact fraction math, the standard library Fraction tool is optional and keeps thirds and sixths precise.

- Fraction (standard library, optional): stores each leftover as a true fraction so the total stays exact.

---

## Inputs

- Starting budget per project: one number, the same budget each project began with. A plain number such as 48.
- Leftover fraction for each project: a top number and a bottom number for each of three projects. For example, three quarters is top 3 and bottom 4.

---

## Steps

### Step 1 — Set up one reusable conversion

Prepare a single way to turn any leftover fraction into the credits a project used. The rule is the same for every project, so define it once and reuse it three times.

### Step 2 — Read the shared budget

Ask for the budget that each project started with. All three projects use this same starting amount.

### Step 3 — Read each project's leftover fraction

Ask in turn for the leftover fraction of the first, second, and third project. For each one, collect a top number and a bottom number. Keep the three pairs separate so each project's leftover stays tied to that project.

### Step 4 — Turn each leftover into the part used

For each project, treat the leftover fraction as a part between zero and one. Subtract that leftover part from one whole to get the part that was used. Take that used part of the budget. Repeat for all three projects.

### Step 5 — Add the three amounts

Add the credits used by the first, second, and third project into one total.

### Step 6 — Tidy the result and report it

If the total comes out to a whole number, show it as a whole number rather than a decimal or fraction. Print the total with text that names the scenario, such as the total credits the three projects used.
